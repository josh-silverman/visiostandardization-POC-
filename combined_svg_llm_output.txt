To change the color of all text to bold dark blue and thicken the lines in your SVG file, you need to modify the `<text>` elements and the stroke properties of the shapes. Below is an example of how you can update your SVG code:

### Original SVG Example:
```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <text x="10" y="20" font-size="16">Sample Text</text>
  <line x1="10" y1="30" x2="100" y2="30" stroke="black" stroke-width="1"/>
  <circle cx="50" cy="50" r="20" stroke="black" stroke-width="1" fill="none"/>
</svg>
```

### Updated SVG Example:
```xml
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- Text updated to bold and dark blue -->
  <text x="10" y="20" font-size="16" font-weight="bold" fill="darkblue">Sample Text</text>
  
  <!-- Lines thickened -->
  <line x1="10" y1="30" x2="100" y2="30" stroke="black" stroke-width="3"/>
  
  <!-- Circle stroke thickened -->
  <circle cx="50" cy="50" r="20" stroke="black" stroke-width="3" fill="none"/>
</svg>
```

### Explanation of Changes:
1. **Text Color and Weight**:
   - Added `font-weight="bold"` to make the text bold.
   - Changed the `fill` attribute to `darkblue` to set the text color to dark blue.

2. **Thickened Lines**:
   - Increased the `stroke-width` attribute for the `<line>` and `<circle>` elements from `1` to `3`.

If you have a more complex SVG file, you can apply these changes to all `<text>` elements and increase the `stroke-width` for all shapes (e.g., `<line>`, `<circle>`, `<rect>`, etc.). Let me know if you need help with a specific SVG!
